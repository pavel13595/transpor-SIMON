### 🚀 Как запустить ПРЯМО СЕЙЧАС:

1. **Простой способ:**
   ```bash
   open index.html
   ```

2. **С локальным сервером:**
   ```bash
   ./start-server.sh
   # Откройте http://localhost:8000
   ```

3. **Через VSCode:**
   - `Cmd+Shift+P` → "Tasks: Run Task" → "Start Development Server"
